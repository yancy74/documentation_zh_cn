= 创建 Kickstart 文件

:context: creating-kickstart-files

您可以使用以下方法创建 Kickstart 文件：


* 复制手动安装后创建的 Kickstart 文件。

* 手动编写整个 Kickstart 文件。请注意，编辑来自其他方法已存在的文件更快，因此不建议使用此方法。

// If needed, you can manually edit the Kickstart file and change the installation options.

注意，某些高级特定的安装选项只能通过手动编辑 Kickstart 文件来配置。
