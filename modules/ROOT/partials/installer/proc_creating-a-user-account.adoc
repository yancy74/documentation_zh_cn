= 创建用户帐户

建议您创建一个用户帐户以完成安装。如果不创建用户帐户，则必须直接以 root 用户身份登录系统，不建议这样做。

.步骤

. 在 *配置* 窗口中，单击 *用户创建*。将打开 *创建用户* 窗口。

. 在 *全名* 字段中输入用户帐户名称，例如：John Smith。

. 在 *用户名* 字段中输入用户名，例如：jsmith。
+
[NOTE]
====
*用户名* 用于从命令行登录；如果安装图形环境，则图形登录管理器使用 *全名*。
====

. 如果用户需要管理员权限，请选中 *使此用户成为管理员* 复选框（安装程序会将用户添加到 wheel 组）。
+
[IMPORTANT]
====
管理员用户可以通过 `sudo` 命令使用用户密码(而不是root密码)执行仅限 root 用户执行的任务。这可能更方便，但也可能造成安全风险。
====

. 选中 *需要密码才能使用此帐户* 复选框。
+
[WARNING]
====
如果您为用户授予管理员权限，请验证该帐户是否受密码保护。在未为帐户分配密码的情况下，切勿授予用户管理员权限。
//TODO: link to some security docs about password security
====

. 在 *密码* 字段中输入密码。

. 在 *确认密码* 字段中输入相同的密码。

.  *保存更改* 以应用更改并返回到 *配置* 窗口。

.  安装过程完成后，单击 *重启* 以重新启动并登录到 {ProductName} {ProductNumber} 系统。
