= 保存屏幕截图

// [[sect-installation-screenshots]]

在图形化安装过程中，您可以随时按 kbd:[Shift + Print Screen] 来捕获当前屏幕。这些屏幕截图保存在 [filename]`/tmp/anaconda-screenshots`。
