= 安装工作流程

此安装工作流程包含在AMD64、Intel 64 和 64位 ARM 架构上使用图形用户界面安装 {productname} 的高级步骤。

.步骤

. 检查系统和硬件要求、下载安装映像文件并创建可引导的安装介质，为安装做好准备。
. 启动安装程序并使用图形用户界面安装 {productname}。
. 完成安装后的任务，例如初始设置和系统注册。

.其他资源

* 有关准备安装的详细信息，请参阅xref:standard-install:assembly_preparing-for-your-installation.adoc[准备安装]。
* 有关启动安装程序的详细信息，请参阅xref:standard-install:assembly_booting-installer.adoc[启动安装]。
* 有关使用图形用户界面安装 {productname} 的详细信息，请参阅xref:standard-install:assembly_graphical-installation.adoc[使用图形用户界面安装]。
* 有关完成安装后任务的详细信息，请参阅xref:standard-install:assembly_post-installation-tasks.adoc[成安装后的任务]。
